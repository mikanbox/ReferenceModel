## How to set up project
- install VS Code Plugin
- ⌘(command) + shift + p : Monkey C: New Project
- ⌘(command) + shift + p : Monkey C: Edit Products
- Open [xxx.mc] file
- ⌘(command) + shift + p : Monkey C: Run Tests
